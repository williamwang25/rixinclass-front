## 总计
- **73个组件**，涵盖了移动应用开发的各种场景需求

## 特点
- 支持 H5、小程序、APP 多平台开发
- 使用最新的前端技术栈（Vue3 + TypeScript）
- 内置约定式路由、layout布局、请求封装等功能
- 无需依赖 HBuilderX，支持命令行开发
- 完善的组件库和工具函数
